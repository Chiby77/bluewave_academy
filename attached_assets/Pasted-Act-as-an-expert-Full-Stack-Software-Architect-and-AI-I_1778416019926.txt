Act as an expert Full-Stack Software Architect and AI Integration Specialist.

I need you to design the architecture, technical stack, and step-by-step implementation plan for an MVP (Minimum Viable Product) of a smart, digital examination and grading system called "The Examinator." The client is Bluewave Academy.

Project Overview:
The Examinator is an end-to-end digital assessment platform. It handles everything from student enrollment and online test-taking to automated AI grading and result distribution. The core AI grading engine will be powered by a Google multimodal LLM (e.g., Gemini 1.5 Pro) capable of processing text, code, and PDF documents.

MVP Core Features & User Stories:

1. User & Class Management (The Foundation)

Roles: Admin/Instructor and Student.

Features: Secure login/authentication. Students must be able to browse and enroll in specific classes or modules. Instructors can manage class rosters.

2. The Digital Assessment Engine (Test-Taking)

Online Quizzes: Multiple-choice and short-answer questions taken directly within the browser.

Coding Practicals: An integrated code editor where students can write, execute, and submit code snippets for practical exams.

Timed Assignments: Instructors can set strict countdown timers for assignments.

File Uploads: Students can upload their written answer scripts (strictly in PDF format) for offline assignments or scanned handwritten exams.

3. The AI Grading Pipeline (The Core Engine)

Processing: When a PDF or code snippet is submitted, the system triggers the Google AI model via API.

Evaluation: The AI evaluates the submission against an instructor-provided grading rubric, answer key, or unit tests (for code).

Scoring: The AI assigns a score and generates constructive, specific feedback for the student based on their errors.

4. Reporting & Notifications (The Output)

Report Generation: The system takes the AI's score and feedback and compiles it into a clean, formatted PDF report.

Email Dispatch: The system automatically emails the final PDF report and the grade to the student's registered email address.
